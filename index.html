Dashboard Data

This dashboard provides pre-computed JSON data for rugby analytics applications.

Available Datasets

  • Summary Statistics
  • Team Offensive Rankings
  • Team Defensive Rankings
  • Player Rankings
  • Upcoming Match Predictions
  • Paths to Victory Analysis
  • Squad Depth Analysis
  • Match Index
  • Player Profiles

Competition Data

League tables and season predictions are organized by competition and season:

  • League Tables: league_table_{competition}_{season}.json
  • Position Probabilities: season_position_probs_{competition}_{season}.json
  • Playoff Probabilities: season_playoff_probs_{competition}_{season}.json
  • Team Heatmaps: team_heatmap_{competition}_{season}.json

About

This dashboard is powered by a Bayesian hierarchical model that tracks player and team performance across rugby union competitions. The model is updated weekly with new match data.

Data sources: URC/Celtic, Premiership, Top 14, European competitions, Six Nations
